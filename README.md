## Requisites

- Node.js;
- Nest.js;

## Como rodar o projeto:

## Instalar

```bash
$ npm install
```

## Rodar no modo desenvolvimento

```bash
# development
$ npm run dev
```
Com isso o servidor será aberto no localhost:3000
